# Internet Banking System

## Overview
A full-stack internet banking application built using Spring Boot and React.

## Features
- User registration
- Deposit money
- Withdraw money
- Transfer funds
- Balance checking
- REST APIs
- Frontend integration

## Tech Stack

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL

### Frontend
- React
- Vite
- Tailwind CSS

### Tools
- Maven
- Git
- GitHub
- Docker
- Postman

## Project Structure

backend/
frontend/

## API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| POST | /users/create | Create user |
| GET | /users/all | Get all users |
| POST | /users/deposit/{id} | Deposit money |
